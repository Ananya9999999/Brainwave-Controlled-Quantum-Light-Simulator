<h1>🧠 Brainwave-Controlled Quantum Light Simulator</h1>


A real-time Brain-Computer Interface inspired simulation that converts brainwave signals into 3D quantum light interference patterns.

This project demonstrates how mental states such as focus and signal amplitude influence quantum-like wave behavior through live visualization.

---

<h2>✨ Features</h2>

- 🧠 Brainwave Signal Simulation
- ⚛️ Quantum Light Interference
- 🌌 3D Quantum Light Visualization
- 📊 Real-Time Animation
- ⚡ Sub-Second Response
- 🧩 Modular Architecture
- 🔬 Research-Inspired Design

---

<h2>🧠 Concept</h2>

The system maps cognitive parameters to wave behavior:

<h3> Brain Parameter     Wave Behavior</h3>

- Signal Amplitude	Wave Intensity
- Mental Focus	    Wave Coherence

<h3>Interpretation</h3>

- Calm → Smooth waves
- Focused → Sharp coherent patterns
- Distracted → Chaotic interference
